# Adult Websites Blocklist

> **Purpose:** This list is intended for use in parental controls, firewall rules, DNS filtering (e.g., Pi-hole, pfSense, OpenDNS), or network-level content blocking.

---

## How to Use This List

- **DNS Filtering:** Add domains to tools like [Pi-hole](https://pi-hole.net/), [AdGuard Home](https://adguard.com/), or [OpenDNS](https://www.opendns.com/).
- **Router/Firewall:** Block domains via your router's parental control settings.
- **Browser Extensions:** Use extensions like uBlock Origin with custom lists.
- **Hosts File:** Add entries as `0.0.0.0 domain.com` in your system's hosts file.
